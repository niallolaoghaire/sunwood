=== WooCommerce Braintree ===
Author: woothemes, skyverge
Tags: woocommerce
Requires at least: 4.1
Tested up to: 4.5.2
Requires WooCommerce at least: 2.4.13
Tested WooCommerce up to: 2.6.0

Adds the Braintree Payment Gateway to your WooCommerce store, allowing customers to securely save their credit card and PayPal account for use with single purchases, pre-orders, subscriptions, and more!

See http://docs.woothemes.com/document/braintree/ for full documentation.

== Installation ==

1. Upload the entire 'woocommerce-gateway-braintree' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
